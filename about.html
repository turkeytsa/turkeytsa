About

This website is an information website about TSA Chapter and CTE Program at our school.
Our website also contains our research about Augmented Reality .
It is written with HTML, CSS and jQuery.
This site is built with Jekyll, which is a a static website generator.
All photos used in this website are either in the public domain or taken by our members.
The template is created by our team.
Our website is hosted on Github
Our team used Media Queries in order to make the website responsive
